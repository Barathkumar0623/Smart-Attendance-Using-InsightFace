# 🧠 AI Smart Attendance System Using InsightFace

## 📌 Project Description
This project is an AI-based Smart Attendance System that uses face recognition technology (InsightFace) to automatically mark attendance and analyze facial expressions.

---

## 🚀 Features
- Face Detection using InsightFace
- Face Recognition for students/employees
- Automatic Attendance Marking
- Unknown Person Detection
- Facial Expression Analysis (Happy, Neutral, Sad, etc.)
- Attendance stored in CSV format
- Flask Web Dashboard for visualization
- Real-time webcam support

---

## 🛠️ Tech Stack
- Python
- OpenCV
- InsightFace
- Flask
- NumPy
- Pandas
